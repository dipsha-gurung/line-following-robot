# Line Following Robot

## Overview

An autonomous line-following robot developed using Arduino Uno, IR sensors, and an L298N motor driver. The robot follows a predefined path by continuously detecting line positions through infrared sensors and adjusting motor movements using embedded control logic.

The project demonstrates practical knowledge of embedded systems, robotics, sensor integration, and Internet of Things (IoT) concepts.

---

## Features

- Autonomous line following
- Real-time IR sensor detection
- Motor speed and direction control
- Embedded control logic using Arduino
- Serial Monitor output for sensor debugging
- IoT-ready architecture for future cloud integration
